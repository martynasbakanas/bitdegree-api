Get started

BitDegree API is currently at version 1.0 and the root endpoint is https://www.bitdegree.org/api/v1.0/ for all resources.

To use this API, you need an API key. API key can be generated on BitDegree website.

get courses

To get courses you need to make a GET call to the following url :
https://www.bitdegree.org/api/v1.0/courses

QUERY PARAMETERS

Field Type Description
api_key String Your API key.
page Integer (optional - default: 1) A numeric value indicating the current page.
page_size Integer (optional - default: 15) A numeric value indicating the page size. Max: 100
instructor_data Boolean (optional) If true, courses will have instructor data
category_data Boolean (optional) If true, courses will have category data

Errors

BitDegree API uses the following error codes:

Error Code Meaning
400 Missing header Accept: application/json
401 Unknown, missing or invalid api_key.
414 Request parameters are too long. Maximum length: 190
429 Daily request limit reached.
500 Server error
